# Niddesh Kumar - Cybersecurity Portfolio

A modern, responsive portfolio website showcasing my cybersecurity expertise, projects, and experience.

## Features

- **Modern Design**: Dark theme with smooth animations and transitions
- **Responsive Layout**: Mobile-friendly and works on all devices
- **Interactive Elements**: 
  - Scroll-triggered animations
  - Hover effects on skills and projects
  - 3D parallax effect on hero image
  - Smooth navigation
- **Professional Content**: 
  - Education and qualifications
  - Technical skills and tools
  - Featured projects and internship experience
  - Contact information

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Responsive CSS Grid and Flexbox
- **Animations**: CSS keyframes and JavaScript Intersection Observer
- **Hosting**: Local development with Python HTTP server
